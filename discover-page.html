Question-1: What are the different ways to select an element in the DOM?

DOM-এ এলিমেন্ট নির্বাচন করার বিভিন্ন পদ্ধতি হলো: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.innerHTML:এটি HTML এলিমেন্টের ভিতরের HTML কোড (যেমন ট্যাগ, উপাদান) সহ সমস্ত কনটেন্ট প্রদান করে। 2. innerText:এটি শুধুমাত্র HTML এলিমেন্টের দৃশ্যমান টেক্সট কনটেন্ট প্রদান করে। এতে সিএসএস বা অন্য কিছু স্টাইলিংয়ের প্রভাব পড়তে পারে। 3. textContent:এটি এলিমেন্টের সমস্ত টেক্সট কনটেন্ট প্রদান করে, তবে এটি innerText এর মতো সিএসএস বা স্টাইলিং এর প্রভাবের সাথে যুক্ত হয় না।

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন হল একটি কৌশল যেখানে প্যারেন্ট এলিমেন্টে ইভেন্ট হ্যান্ডলার যুক্ত করা হয়, এবং সেই ইভেন্ট প্যারেন্ট এলিমেন্টের মাধ্যমে চাইল্ড এলিমেন্টগুলিতে প্রভাবিত হয়। এর মাধ্যমে, প্যারেন্ট এলিমেন্টই চাইল্ড এলিমেন্টের ইভেন্ট হ্যান্ডল করে। এই পদ্ধতি অনেক চাইল্ড এলিমেন্টে আলাদা করে ইভেন্ট অ্যাটাচ না করে, একটি ইভেন্ট হ্যান্ডলার দিয়ে কাজটি সম্পন্ন করে।

Question-4: What is event bubbling in the DOM?

ইভেন্ট বাবলিং হল একটি DOM ইভেন্টের আচরণ যেখানে একটি ইভেন্ট প্রথমে সেই এলিমেন্টে ট্রিগার হয়, যেখানে ইভেন্টটি ঘটেছে, এবং তারপর ধীরে ধীরে প্যারেন্ট এলিমেন্ট থেকে প্যারেন্ট এলিমেন্টে বুদ্বুদ (bubble) করতে থাকে যতক্ষণ না এটি রুট এলিমেন্ট (যেমন document বা window) পৌঁছায়।

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript ব্যবহার করে DOM এ উপাদান তৈরি, যোগ, এবং মুছে ফেলার পদ্ধতি:
1. তৈরি করা: document.createElement() দিয়ে নতুন উপাদান তৈরি করা হয়।
2. যোগ করা: appendChild() দিয়ে নতুন উপাদান যোগ করা হয়।
3. মুছে ফেলা: remove() অথবা removeChild() দিয়ে উপাদান মুছে ফেলা হয়।
